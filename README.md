# 🎨 Inside 3D ART

Tienda online de impresiones 3D personalizadas. Proyecto final para CoderHouse JavaScript.

## 📁 Estructura del proyecto

```
Proyectofinal-JMM/
├── index.html              ← Página principal
├── css/
│   ├── styles.css          ← Estilos globales
│   └── pages.css           ← Estilos páginas internas
├── js/
│   └── app.js              ← Lógica principal JS
├── data/
│   └── products.json       ← Datos de productos (JSON local)
├── img/
│   ├── logo-Inside.png
│   ├── whatsapp.png
│   ├── instagram.png
│   ├── gmail.png
│   ├── ubicacion.png
│   ├── Diseño_jarratermica_afa.jpg

│   ├── Lampara_velador_boca.jpg

│   ├── Diseño_luna.jpg
│   ├── mini-dragon.webp

│   └── ... (más imágenes de productos)
└── pages/
    ├── nosotros.html
    ├── preguntas.html
    ├── terminos.html
    └── politicas.html
```

## 🚀 Cómo abrir el proyecto

1. Cloná el repositorio
2. Abrí `index.html` en tu navegador  
   ⚠️ Para que el JSON local funcione, usá **Live Server** en VS Code

## ✅ Funcionalidades

- Catálogo de productos cargados desde `products.json` (async/await)
- Filtro por categoría (chopp, lámparas, personalizados)
- Búsqueda en tiempo real
- Carrito con localStorage
- Descuento automático del 10% en compras mayores a $60.000
- Modales custom (sin alert/prompt/confirm)
- Notificaciones con **SweetAlert2**
- **FakeStore API** → sección Ofertas Especiales
- Testimonios con nombres de clientes reales
- Páginas: Nosotros, Preguntas Frecuentes, Términos y Condiciones, Políticas de Envío

## 📚 Tecnologías

- HTML5 + CSS3 (variables, grid, flexbox, parallax)
- JavaScript ES6+ (async/await, fetch, localStorage)
- SweetAlert2
- FakeStore API

## 🛍️ Categorías de productos

- **🍺 Jarra 3D Chopp 500ml** — Diseños temáticos (AFA, Fernet Branca, bandas musicales)
- **💡 Lámparas veladores 3D** — Luna, Mesa Nórdica, escudos de equipos de fútbol
- **😎 Personalizados** — Figuras, llaveros, piezas de ajedrez y más
